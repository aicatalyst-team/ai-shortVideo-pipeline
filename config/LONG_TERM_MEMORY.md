# myAiVideos Long-Term Memory

## Handwritten Permanent Rules

1. 解说类短视频优先做图文混剪，不追求复杂电影叙事。
2. 前 3 秒必须给出冲突、悬念、数字或强情绪，不铺垫背景。
3. 每条视频只讲一个核心观点，避免在 30 秒内塞多个主题。
4. 热搜解说要先交代为什么值得看，再解释发生了什么。
5. 知识科普要用生活例子开场，再给结论和解释。
6. 情感故事要少讲大道理，多给具体场景和细节。
7. 奇闻趣事要保留悬念，但不能牺牲事实清晰度。
8. 社会观察要敢于提出判断，但避免空泛站队。
9. 字幕必须服务信息节奏，不要遮挡主体和关键画面。
10. 任何自动提炼的规律都必须先过人工审核，再进入生成 prompt。
